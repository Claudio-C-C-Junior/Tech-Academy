UX do Projeto

Persona

O público-alvo do jogo é formado por jovens e adultos que gostam de jogos de plataforma simples, com comandos e visuais simples.

Problema que o site resolve

O site apresenta o projeto Fortuna Roubada, explica sua proposta, mostra imagens do jogo, permite contato com o desenvolvedor e oferece uma área para jogar diretamente no navegador.

Decisões visuais

Foi utilizada uma identidade visual com cores fortes, como preto, vermelho, amarelo e branco, para combinar com a proposta de ação do jogo. Os botões foram destacados para facilitar a navegação.

Usabilidade

O menu principal está presente em todas as páginas, os botões têm nomes claros e o site foi adaptado para funcionar também em telas menores, deixando o conteúdo organizado no celular.

Acessibilidade

As imagens possuem texto alternativo, os botões possuem contraste visível e o conteúdo foi organizado com títulos e parágrafos para facilitar a leitura.
